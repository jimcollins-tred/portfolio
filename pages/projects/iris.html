The Iris is a responsive, full-stack news site built with React. It allows users to browse, comment and vote upon numerous topics and articles, all within a clean, attractive design.

The project was completed towards the end of my time at Northcoders bootcamp. It had two stages: a back-end phase in which I set up the database and back-end api; and a front-end phase in which I designed and built a React app that made cals to this api and displayed the results.

This front-end is fully responsive, with specific UI elements and menus for tablets and phones. The most popular articles appear in a sidebar to drive site engagement, and articles can be sorted in various ways.

The back-end api was driven by Express server, which I accessed via Axios calls from the front-end. I used Knex for database seeding, migrations and queries, and postgreSQL for the database itself.

Users can perform a fuzzy search of topics, which will return any appropriate matches. They can then display the articles solely for one topic.

User login

When users log in to the site, they gain access to buttons allowing them to post a new article, delete their own articles, or comment on existing articles.

If no user is logged in, these buttons are not rendered.
