The Iris is a responsive, full-stack news site completed towards the end of my time at Northcoders bootcamp. It allows users to browse topics and articles, comment on articles, vote on articles and comments, and post their own articles, all within a clean, attractive design.

The project had two stages: a back-end phase in which I set up the database and back-end api; and a front-end phase in which I designed and built a React app that made cals to this api and displayed the results.

This front-end is fully responsive, with specific UI elements and menus for tablets and phones. The most popular articles appear in a sidebar to drive site engagement, and articles can be sorted in various ways.

The back-end api was driven by Express server, which I accessed via Axios calls from the front-end. I used Knex for database seeding, migrations and queries, and postgreSQL for the database itself.
