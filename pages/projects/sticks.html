Sticks + Stones is a web-based hangman game built in React. Players can guess from thousands of horror film titles pulled from The Movie Database, all within an attractive, fun UI that leans heavily into the horror aesthetic.

The core game logic of this project was developed while I was on the Northcoders bootcamp, pair programming with my partner Matthew Beadman. We also set up the React structure, using Reach Router to enable navigation.

After graduation, I decided to develop the game as an enjoyable way of learning new technologies.

Developing the game

The app now features numerous animated elements for a slick, game-like feel, powered by React Transition Group. It also includes an animated hangman built around the HTML canvas.

Rather than hard-coding a bank of phrases, I now make Axios requests to the Movie Database API. This provides a huge supply of phrases players can guess.

I also made use of the Styled Components library for more intuitive handling of my CSS.

Project / Spotlight

The Hangman

The game uses a HTML canvas to display the hangman. To animate the drawing, I built an animated canvas component that receives draw instructions and renders them a frame at a time, using JavaScriptâ€™s requestAnimationFrame to prompt recursive calls to the render function.

The hangman component itself contains an array of draw instructions, one for each stage of the hangman. These are passed in turn to the canvas to be drawn, as each incorrect guess is made.

Animation

React Transition Group provides a lightweight API for handling component state over time. By monitoring items in React state, it informs our components when the items' states change and allows us to animate accordingly. For example, removing the placeholders and animating the display of letters in the phrase when they are correctly guessed.

{}

Film Title Retrieval

An initial array of 20 titles is retrieved from the Movie Database API, which are then filtered to remove titles that are excessively long or contain numbers. Random numbers are generated to determine both the page of titles retrieved and the title selected from the retrieved array. Once used, film titles are removed from the array to remove any chance of them appearing again within a session. Only when these 20 titles are all used do we retrieve another 20.
